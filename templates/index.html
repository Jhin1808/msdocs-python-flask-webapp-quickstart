About this website

This website is my Program #4 project: Website + Storage. It utilizes cloud services to create a website with specific functionalities integrated with cloud storage and NoSQL databases

  • The website should have a "Load Data" button to import data from a provided URL into cloud storage and a NoSQL database.
  • A "Clear Data" button should remove data from both the cloud storage and NoSQL databas
  • Users should be able to input a first name and/or last name and retrieve corresponding data from the NoSQL database using a "Query" button
  • Data to be loaded will be in text format, with each line containing a last name, a first name, and a series of key-value pairs as attributes
  • The website must handle data loading intelligently, adding new items and updating existing ones in the NoSQL database without removing previous data
  • The system must support querying by last name and/or first name, with the assumption that each name combination is unique
